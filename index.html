Studio Notes & Trends

这里可以放行业热点、设计趋势、小红书内容观察、AI 工具更新。它负责提升网站的专业感，不直接承担成交。

Trend

AI Resume Tools Are Becoming Workflow Products

简历工具的竞争点不只是 AI 文本生成，而是信息填写、岗位匹配、模板排版和导出流程的整合。

Industry Note

从“AI润色”到“一页完成”的简历产品体验

用户可以自己去 AI 优化文字，但真正省时间的是：内容直接映射到固定排版、自动预览、一键导出。

Visual

Low Saturation Visual Assets Are Easier to Reuse

低饱和、留白、手绘感、非 AI 感素材，更适合被设计师二次编辑和长期复用。

Design Trend

为什么“干净的素材库”比花哨效果更耐用

素材产品最重要的是可编辑、可组合、可复用。网站也应保持清楚分类，而不是一次性堆满效果。

Prompt

Prompt Libraries Need Structure, Not Just Keywords

好的 prompt 库应该按照构图、风格、色彩、细节、氛围、输出用途来组织。

Prompt System

Prompt 生成器可以成为设计资源的一部分

把你长期积累的建筑插画、拼贴风、活动分析图关键词，变成可以搜索、复制、复用的内容库。
